- 問58. 以下のコードを実行すると画面に何と表示されるか教えてください。
    ```java
    public class T58 {
        public static void main(String[] args) {
            int[] a = new int[4];
            for(int i=0; i<a.length; i++) {
                a[i] = a.length - i;
            }
            System.out.println(a[3]);
        }
    }
    ```
    - `1`
